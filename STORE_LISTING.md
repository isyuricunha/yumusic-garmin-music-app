## Additional Hardware Requirements

**Does this app require additional hardware?** Yes

**Product URL:**
```
https://www.navidrome.org/docs/installation/
```

**Description:**
YuMusic requires a Navidrome, Subsonic, AirSonic, or Gonic music server. Users must set up their own server or have access to an existing one. The server can be self-hosted on:
- Home server / NAS
- Cloud VPS
- Docker container
- Any computer with network access

Free and open-source server options:
- Navidrome: https://www.navidrome.org/
- Gonic: https://github.com/sentriz/gonic
- AirSonic: https://airsonic.github.io/
