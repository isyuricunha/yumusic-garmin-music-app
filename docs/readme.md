# YuMusic Documentation

Welcome to the official documentation for **YuMusic**, a sleek, Spotify-like music player for Garmin watches that connects to your personal home music server.

## Overview

YuMusic acts as an Audio Content Provider on Garmin Connect IQ. It syncs playlists and tracks directly from a Subsonic-compatible server (like Navidrome) over Wi-Fi, allowing you to run and work out without your phone.
